# Library Management System (C++)

A simple console-based Library Management System developed using C++ and file handling.

## Features

* Add Book
* Search by Title
* Search by Author
* Issue Book
* Return Book
* Display Books
* Persistent Data Storage

## Technologies Used

* C++
* OOP
* File Handling

## Run the Project

```bash id="n7r2kx"
g++ librarymanagementsystem.cpp -o main
.\main.exe
```
